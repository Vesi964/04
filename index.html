📊 Функции с параметри - Всички функции получават информация чрез параметри

power(name, cost, color, icon)

Дефинира способност със параметри: име, цена, цвят и икона

Пример: power("Огнена топка", 30, "fire", "🔥")
calculateDamage(baseDamage, energyUsed, multiplier)

Изчислява урон - параметри: базов урон, енергия, множител

Пример: calculateDamage(45, 30, 1.3) → 73
animateEffect(effectType, duration, intensity)

Управлява ефекти - параметри: тип, време, интензивност

Пример: animateEffect("fire", 800, 0.8)
formatLogMessage(action, abilityName, value)

Форматира съобщения - параметри: действие, име, стойност

Пример: formatLogMessage("damage", "Огнена топка", 45)
checkAbilityStatus(current, max, cost, cooldown)

Проверява статус - параметри: енергия, макс, цена, cooldown

Пример: checkAbilityStatus(100, 100, 30, 0) → "available"
getEnergyPercentage(current, max, decimals)

Калкулира процент - параметри: текуща, макс, десетични

Пример: getEnergyPercentage(50, 100, 2) → 50.00%
createAbility(id, name, cost, icon, effect, cooldown, multiplier)

Генерира способност - всички параметри задават способността

Пример: createAbility("1", "Огнена топка", 30, "🔥", "fire", 3000, 1)
triggerAbilityEffect(ability, abilityId)

Метод - параметри: обект на способност, ID

Пример: this.triggerAbilityEffect(ability, 1)
triggerColorEffect(ability, abilityId)

Метод - параметри: обект на способност, ID

Пример: this.triggerColorEffect(ability, 2)
showAbilityNotification(icon, name, cost)

Метод - параметри: икона, име, цена на способност

Пример: this.showAbilityNotification("🔥", "Огнена топка", 30)
startCooldown(abilityId, cooldownTime, abilityName)

Метод - параметри: ID, време, име на способност

Пример: this.startCooldown(1, 3000, "Огнена топка")
